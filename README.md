# Glacier
### A simple statically typed language written in C++.
---
## Installation.
```
git clone https://github.com/harrykearney/Glacier.git
cd Glacier
./glacier.exe main.xg
```

## Hello, World!
```
PRINTLINE "Hello, World!"
```

## Hello, World! Pt.2 
```
PRINT "Hello, World!"
NEWLINE
```

## Math Evaluation.
```
EVAL 2 + 2 - (4 - 1)
```
